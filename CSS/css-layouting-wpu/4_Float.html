Float

Deskripsi

Float merupakan properti pada CSS untuk mengatur posisi sebuah elemen.
Sebuah elemen dapat dipaksa untuk berada di sebelah kiri atau kanan dari parent/ pembungkusnya dengan menggunakan properti ini.

https://css-tricks.com/all-about-floats/


Properti pada float

  • none
  • left
  • right

Yang bisa dilakukan menggunakan properti float :

text wrapping : membuat text mengelilingi gambar/elemen lain
image gallery : membuat serangkaian gambar menjadi galeri
multi-column layout : membuat halaman memiliki beberapa kolom
dll : dan masih banyak lagi
